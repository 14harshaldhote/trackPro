Help & Getting Started Guide

Everything you need to know about Tracker Pro

Quick Start Guide

1. Create a Tracker

Click "New Tracker", give it a name, and choose a template category (Productivity, Fitness, etc.)

2. Add or Edit Tasks

Go to "Manage Tasks" to add, edit, or remove activities you want to track

3. Track Daily

Use "Today" view to mark tasks as done. Click status buttons to cycle: TODO → IN PROGRESS → DONE

4. Review Insights

Check Week/Month views for patterns, and Insights for personalized recommendations

Time-Based Views

Today View

Focus on what you need to do TODAY. Shows all tasks with current status, progress bar, and completion stats.

  • Click status circles to toggle: TODO → IN PROGRESS → DONE
  • Tasks strike through when completed
  • Real-time progress updates

Week View

7-day grid (Mon-Sun) showing all tasks across the week. Perfect for seeing patterns and consistency.

  • Each cell = one task on one day
  • Click any cell to toggle status
  • Today's column is highlighted
  • Navigate previous/next weeks

Month View

Full calendar grid with all tasks. Best for long-term habit tracking.

Custom Range

Select any date range for flexible reporting. Includes bulk actions.

Understanding Task Status

TODO Grey circle ○ Not started
IN PROGRESS Blue half-circle ◐ Currently working on it
DONE Green checkmark ✓ Completed
MISSED Red X ✗ Overdue/skipped
BLOCKED Orange warning ⚠ Can't proceed (optional)

Frequently Asked Questions

How do I add tasks to a tracker?

Go to your tracker page → Click "Manage Tasks" → Click "+ Add New Task". Fill in the task description and category.

Can I use templates?

Yes! When creating a tracker, select a category (Productivity, Fitness, Study, Wellbeing) and tasks will be auto-added. You can also browse all templates from the sidebar.

What's the difference between deleting a template vs. instances?

Deleting a template removes it from future days. Deleting instances removes all past data for that task. By default, only the template is deleted to preserve your history.

How do bulk actions work?

In Custom Range view, you can select multiple tasks and mark them all as done, or use "Mark Overdue as Missed" to automatically update past TODO tasks.

Where can I see my progress?

Check the "Insights" page for plain-English analysis, recommendations, and per-task performance. Analytics page has advanced charts and correlations.

Pro Tips

  • Start with 3-5 core tasks, not 20 - you'll have better follow-through
  • Use the Week view to spot patterns in when you complete tasks
  • Weight important tasks higher (1-10 scale) - it affects your completion percentage
  • Check Insights weekly for personalized recommendations
  • Duplicate tasks to quickly create similar ones
